# Personalized Learning Recommendation System

## Project Overview

This project is a machine learning-based course recommendation system developed using Python. The system recommends relevant online courses to users based on the course they are interested in. The recommendation process is performed using Natural Language Processing (NLP) techniques and similarity analysis.

The project uses the Coursera dataset and recommends courses that have similar skills, difficulty levels, and learning content.

---

## Objective

The main objective of this project is to help learners discover courses related to their interests and learning goals. By analyzing course information, the system provides personalized recommendations that can improve the learning experience.

---

## Dataset

Dataset Used: Coursera Course Dataset

Important attributes used in this project:

* Course Name
* Skills
* Level
* Partner/Organization

These attributes are combined to create a feature set that represents each course.

## Methodology

### 1. Data Preprocessing

* Loaded the Coursera dataset using Pandas.
* Handled missing values in both text and numeric columns.
* Selected relevant attributes for recommendation.

### 2. Feature Engineering

The following columns were combined into a single feature:

* Course
* Skills
* Level
* Partner

This combined feature helps the model understand the overall characteristics of a course.

### 3. TF-IDF Vectorization

TF-IDF (Term Frequency-Inverse Document Frequency) was used to convert textual course information into numerical vectors.

### 4. Similarity Calculation

Cosine Similarity was used to measure the similarity between courses.

Courses with higher similarity scores are recommended to the user.

### 5. Recommendation Generation

When a user enters a course name:

1. The selected course is identified.
2. Similarity scores are calculated.
3. Top matching courses are retrieved.
4. Recommendations are displayed.

---

## Evaluation Metrics

The recommendation system was evaluated using the following metrics:

* Precision@10
* Recall@10
* F1-Score@10
* Mean Average Precision (MAP)
* Normalized Discounted Cumulative Gain (NDCG)

These metrics help measure the relevance and quality of the recommendations generated by the system.
